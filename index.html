Workflow

  1. Create a folder: /papers/<slug>/.
  2. Add index.html, styles.css, script.js.
  3. Link it on this index page.
  4. Push to GitHub → Vercel deploys.

流程

  1. 创建文件夹：/papers/<slug>/。
  2. 添加 index.html、styles.css、script.js。
  3. 在索引页加入链接。
  4. 推送到 GitHub → Vercel 自动部署。

About WAP

WAP is a static site. Every paper is a standalone web page, so you can freely design layouts or add interactive demos per paper.

关于 WAP

WAP 是纯静态站点。每篇论文都是独立页面，因此你可以自由设计布局或加入交互演示。
