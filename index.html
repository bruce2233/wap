Bilingual HS + Grad GitHub Workflow

Quick actions

  • Use search to find a paper page.
  • Drop a JSON file to add a new paper.
  • Share: wap.bryan2233.top/<slug>

快速入口

  • 搜索论文页面。
  • 新增一个 JSON 文件即可发布。
  • 分享格式：wap.bryan2233.top/<slug>

Workflow

  1. Add a new JSON file in data/papers/.
  2. Update data/papers.json (or run the script).
  3. Push to GitHub → Vercel deploys automatically.

流程

  1. 在 data/papers/ 新建论文 JSON。
  2. 更新 data/papers.json（或运行脚本）。
  3. 推送到 GitHub → Vercel 自动部署。

About WAP

WAP is a static site with a client-side router. It renders bilingual, dual-level explanations from JSON. The same page supports both high school and graduate-level readers.

关于 WAP

WAP 是纯静态站点 + 前端路由，使用 JSON 渲染中英双版、双层级讲解。

WAP — Web Any Paper. Built for bilingual sharing and layered understanding.
